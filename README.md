# 📊 Total Sales Analysis by Month – Excel Pivot Dashboard

This project provides a clear and interactive analysis of monthly **sales**, **profit**, and **units sold** using Excel's Pivot Table and Pivot Chart tools.

## 🔍 Overview

- **Goal**: Identify trends in total sales, profits, and units sold across a 12-month period.
- **Tool used**: Microsoft Excel (Pivot Tables + Pivot Charts)
- **Dataset**: `Shopora_Orders` – fictional eCommerce sales data

The dashboard helps answer:
- Which months had the highest and lowest total sales?
- How do units sold relate to profit trends?
- Are there any significant outliers or dips in performance?

## 📁 Features

- 📆 **Dynamic Pivot Table**: Aggregated by month
- 📈 **Line Chart Visualization**: Compares three key metrics side-by-side
- 🧮 Metrics used:
  - Sum of `Total_Sales`
  - Sum of `Profit`
  - Sum of `Units_Sold`

## 📌 Key Insights

- **June 2024** shows a major drop in all three KPIs — a strong outlier worth investigating.
- **May 2025** had the highest spike in total sales across the year.
- Overall performance remained relatively stable with visible seasonal patterns.

## 🧰 How It Works

1. Imported and cleaned the `Shopora_Orders` dataset
2. Created a Pivot Table with:
   - Rows: `Month`
   - Values: Sum of `Total_Sales`, `Profit`, `Units_Sold`
3. Built a Pivot Chart from the table using a **line chart** for multi-metric trend analysis
4. Formatted the chart for clarity and visual appeal

- Data visualization best practices
- Dashboard design in Excel
